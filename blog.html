What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface that represents a webpage as a hierarchical tree of nodes, allowing dynamic manipulation of its structure and content. It enables scripts to interact with HTML and XML documents in real-time.

How do you select an element from the DOM?

I can select a DOM element by using document.getElementById() for IDs or document.querySelector() for more flexible CSS selectors. These methods let me easily access and manipulate specific elements on a webpage.

What is event delegation in the context of the DOM, and why is it useful?

Event delegation is a technique where a single event listener is added to a parent element to manage events for its child elements, using event bubbling. It improves performance and simplifies code by reducing the number of event listeners needed.

How do you manipulate an element's attributes and styles using the DOM?

I can manipulate an element's attributes using element.setAttribute() or directly by element.attributeName, and I can change its styles with element.style.propertyName. This allows me to dynamically update elements' appearance and behavior.
